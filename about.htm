RMusic Music Player
This song player has been developed by Reuben Mark Elias, an undergraduate pursuing BTech in Computer Engineering from Delhi Technological University as a solo project.
  • The music player provides the user with all the necessary functionalities like play, pause, rewind, fast-forward. The user is also allowed to seek the song to any desired instant with the help of the seekbar. The user is kindly advised to keep the device connected to the internet throughout his use of this website as some functionalities require internet resources.
  • Some songs have been hardcoded in the program for testing purposes.
  • Linking the player using an API with a copyright friendly website can further improve user-experience.
